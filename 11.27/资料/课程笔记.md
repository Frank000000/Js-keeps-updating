# 运算符
  * 算术运算符
    - 加(+)   
      ```js
      (1) 10 + 20;
      (2) "i" + "see" + "u";
      (3) 10 + "1203" + 10 + true + null 
      ```
    - 减(-)
    - 乘(*)
    - 除(/)
    - 模(%) `符号`

  * 关系运算符

    `符号`：`> < >= <= != == ===`

    `重点`：相等和全等

    ```js
    //相等(==)
    console.log(2 == "2"); //true
    //全等(===)
    console.log(2 === "2"); //false
    ```

  * 逻辑运算符

    `逻辑非` `!操作数` 

    `逻辑与` `操作数1 && 操作数2`

    `逻辑或` `操作数1 || 操作数2`

  * 赋值运算符

    `符号： += -= *= /= %=`

  * 自增自减运算符

    `①  ++i --i`

    `②  i++ i--`
  
  * 三元运算符

    `符号：操作数1(条件表达式) ？ 操作数2 ： 操作数3`
